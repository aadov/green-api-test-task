# GREEN-API Test Task

Тестовое задание на должность “Специалист технической поддержки 2 линии”.

## Реализовано

HTML-страница с вызовами методов:

- getSettings  
- getStateInstance  
- sendMessage  
- sendFileByUrl  

Все ответы выводятся в отдельное поле на странице.

## Технологии

- HTML  
- CSS  
- JavaScript  

## Ссылка на проект

👉 https://aadov.github.io/green-api-test-task/

## Примечание

На этапе выполнения не удалось получить доступ к GREEN-API (не приходил код авторизации), поэтому реализована имитация API.

Структура кода позволяет легко подключить реальный API без изменений интерфейса.

## Как запустить

Открыть файл `index.html` в браузере  
или использовать Live Server в VS Code
